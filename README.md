# FoodGo

A Flutter food delivery application project.

## About this Project

This project was built primarily to practice and explore **UI design** in Flutter. It includes:
- A clean and modern user interface for a food delivery app.
- Implementation of basic application logic.
- Hands-on learning and implementation of **Riverpod** for state management.

## Features (UI Focus)
- Splash Screen
- Home Page with search and categories
- Item Details with customization options
- Favorite management
- Profile and Customer Support placeholders
